## How to Capture

1. Open `https://realestate-mcp.jp/dashboard.html`
2. Use browser DevTools to set viewport to 1280x800
3. Capture with `Cmd+Shift+4` (macOS) or `Win+Shift+S` (Windows)
4. Save as PNG in this directory
