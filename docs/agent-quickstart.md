# 不動産業者向けクイックスタートガイド

**Japan Real Estate Intel** を 3 分で始められるガイドです。

> **デモ・プレゼンでは [free-demo-prompts.md](./free-demo-prompts.md) の3本だけ**を使ってください。PDF・リニア数値シミュ・契約支援は **Pro** で、Free 接続ではツールが拒否されます。

---

## ステップ 1: ダッシュボードを開く

ブラウザで以下にアクセスしてください:

```
https://realestate-mcp.jp/dashboard.html
```

iPad / スマートフォンの場合は、ホーム画面に追加するとアプリのように使えます:
- **iPhone / iPad**: Safari の共有ボタン → 「ホーム画面に追加」
- **Android**: Chrome の「…」→「ホーム画面に追加」

## ステップ 2: 都道府県・市区町村を選択

左パネルの「都道府県」から対象エリアを選択 → 地図が更新されます。
対応エリア: 愛知・東京・大阪・福岡・北海道・神奈川・京都・兵庫・埼玉・千葉

## ステップ 3: ダッシュボードのモード

| モード | URL | 用途 |
|---|---|---|
| 投資分析（デフォルト） | `/dashboard.html?prefecture=aichi` | 地価・リスク・人流を総合分析 |
| 店舗出店 | `?mode=store` | 商業・人流レイヤー重視 |
| キャッシュフロー概算 | `?mode=cashflow` | レバレッジCFプレビュー（詳細は MCP ツール `simulate_leveraged_cashflow`） |

リノベ利回り・契約支援の **高度な出力** は MCP ツール（多くは **Pro**）から実行します。一覧は [free-demo-prompts.md](./free-demo-prompts.md)。

## 4 つの体験動線

### 1. 投資判断

ChatGPT・Claude・Cursor に以下のように聞くと、AIが自動で分析ツールを呼び出し、ダッシュボードが会話内に表示されます:

```
名古屋市中区の不動産投資機会を分析して。
リニア中央新幹線の影響も含めて教えて。
```

ChatGPT ではさらにこんな質問も有効です:

```
東京都と大阪府を投資スコアで比較して、レーダーチャートで見せて
```

```
愛知県で割安シグナルのあるエリアを価格トライアングルで探して
```

最短で地図から始めたい場合は、次の一言で十分です:

```
名古屋で買い場になりそうなエリアをChatGPT内の地図とグラフで見せて
```

表示後はダッシュボード内の「ChatGPTで次にする」から、深掘り・価格三角・比較・レポート化へ進めます。

### 2. リノベ利回り

ダッシュボードの `?mode=renovation` を開き:
1. 区（例: 中区）・町丁目（例: 栄）を選択
2. 築年数・面積・物件種別を入力
3. 「利回り計算」ボタンをクリック

表面利回り・実質利回り・リノベ費用のレンジが即座に表示されます。

### 3. 売買契約支援

ダッシュボードの `?mode=contract` を開き:
1. 区・町丁目・価格・築年数を入力
2. 「契約支援パッケージ生成」をクリック

リスクマトリックス・価格交渉アンカー・推奨特約が生成されます。
「リスク評価のみ」で簡易チェックも可能です。

### 4. 現地下見

iPad でダッシュボードをホーム画面に追加して現地へ:
- 地図上で物件周辺の地価・人流・災害リスクをリアルタイム確認
- QR コードを共有してお客様にも見せられます

## よくある質問

**Q: 無料で使えますか?**
A: 基本ツール（地価分析・リスク評価・地価予測など）は無料です。PDF レポート・リノベ利回り・契約支援などの高度な機能は Pro プランで利用できます。

**Q: データの更新頻度は?**
A: 公示地価は年 1 回（3 月）、取引データは四半期、人口統計は国勢調査（5 年周期）に基づいています。e-Stat・MLIT API 経由で最新データを取得しています。

**Q: 対応エリアは?**
A: 愛知県（名古屋市全16区 + 主要都市）、東京都、大阪府、福岡県、北海道、神奈川県、京都府、兵庫県、埼玉県、千葉県の 10 都道府県に対応しています。

**Q: AI チャット（Claude / ChatGPT）から使うには?**
A: MCP サーバーとして接続します。Claude / Cursor / ChatGPT いずれでもインタラクティブな地図・グラフ・価格パネルが会話内に直接表示されます（MCP Apps 対応）。設定方法は `docs/claude-desktop-setup.md` または `docs/chatgpt-integration.md` を参照してください。

**Q: サポートはどこに問い合わせますか?**
A: GitHub Issues または運営にお問い合わせください。
