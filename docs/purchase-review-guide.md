# 購入候補物件レビューガイド（経営者向け）

不動産屋の経営者が物件の購入を勧められた時に、その情報を読み込んで、数値に基づいて的確なレビューをするための意思決定支援ツールです。

## 概要

`review_purchase_recommendation` ツールは、販売図面や物件資料から得られる情報を入力することで、以下の5軸で物件を評価します：

1. **価格評価** — 販売価格 vs 公示地価・路線価・取引相場の乖離を分析
2. **利回り評価** — 表面利回り・実質利回り・投資回収年数を算出
3. **リスク評価** — 空室率・老朽化・修繕リスクを総合評価
4. **将来性評価** — 構造・推奨理由から将来性を評価
5. **契約条件評価** — 融資期間・瑕疵担保・引渡し条件・既存賃貸借を審査

総合スコアと判断（購入推奨 / 価格交渉推奨 / 保留 / 非推奨）を返却し、レッドフラグ・交渉ポイント・推奨特約条項も提示します。

## 使い方

### 基本的なプロンプト例

```
この物件の購入を検討している。以下の情報でレビューしてほしい：

- 所在地: 愛知県名古屋市中区錦3-1-1
- 物件種別: mansion
- 売出価格: 45000000
- 専有面積: 65.5
- 築年数: 12
- 構造: RC
- 現況年間賃料: 2400000
- 想定年間賃料: 2640000
- 想定稼働率: 0.92
- 参考公示地価: 680000
- 参考路線価: 520000
- 近隣取引相場中央値: 610000

仲介会社は「駅近で賃貸需要が安定している」と勧めている。
```

### 契約条件を指定したプロンプト例

```
上記の物件について、以下の契約条件で審査してほしい：

- 融資期間: 45日
- 建物検査: 実施済み
- 瑕疵担保: 24ヶ月
- 既存賃貸借: あり
- 中途解約違約金: あり

この条件で購入すべきか、交渉すべきか判断してほしい。
```

## 出力の見方

### 総合判断

- **buy（購入推奨）**: 総合スコア80点以上。割安で利回りも安定。
- **negotiate（価格交渉推奨）**: 総合スコア65-79点。価格交渉の余地あり。
- **hold（保留）**: 総合スコア45-64点。追加情報が必要または条件交渉が必要。
- **reject（非推奨）**: 総合スコア44点以下。レッドフラグ多数。

### 5軸スコア

各軸のスコア（0-100点）と根拠が表示されます。40点未満の軸は要注意です。

### レッドフラグ

重大なリスク要因（割高・低利回り・老朽化・赤字運営など）がリストアップされます。

### 交渉ポイント

価格交渉・条件交渉の材料となるポイントが提示されます。

### 推奨特約条項

契約時に追加すべき特約条項とその優先度（must / recommended / optional）が表示されます。

## レバレッジ10年シミュレーション

銀行借入を使う案件では、`simulate_leveraged_cashflow` で10年の年次収支を確認できます。利率、LTV、返済期間、賃料、空室率、経費、固定資産税、減価償却年数、出口キャップレートを入力すると、NOI、返済額、税引後キャッシュフロー、DSCR、IRR、Equity Multiple、金利・空室率の感応度を返します。

### プロンプト例

```
この物件を銀行借入ありで10年シミュレーションして。

- 所在地: 愛知県名古屋市中区錦三丁目
- 物件種別: mansion
- 購入価格: 48,000,000円
- 初期取得費用: 2,000,000円
- リノベ費用: 1,000,000円
- 年間賃料: 3,000,000円
- その他収入: 120,000円/年
- 空室率: 6%
- 年間運営費: 520,000円
- 固定資産税等: 180,000円/年
- 年間CAPEX: 120,000円
- 借入: LTV 70%、年利2.1%、返済期間25年、元利均等
- 前提: 賃料成長1%、経費上昇1.5%、10年目売却、出口キャップ5%、売却費用3%、税率20%

10年の年次表、DSCR、税引後CF、IRR、金利上昇時の感応度を見せて。
```

### 経営判断で見るべき数字

- **DSCR**: 1.1倍未満は返済余力が薄い。銀行交渉やLTV引き下げを検討。
- **税引後CF**: 赤字年がある場合、空室率・修繕費・金利を保守的に見直す。
- **IRR / Equity Multiple**: 出口キャップレートを変えて、売却価格依存度を確認。
- **感応度**: 金利+0.5%/+1.0%、空室率上昇で投資判断が崩れないか確認。

## 情報元

- 公示地価: 国土交通省「地価公示」
- 路線価: 国税庁「財産評価基準書 路線価図」
- 取引価格情報: 国土交通省「不動産取引価格情報検索」
- 人口推計: 総務省「国勢調査」「人口推計」
- 空き家率: 総務省「住宅・土地統計調査」
