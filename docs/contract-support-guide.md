# 売買契約支援ガイド (v6.9.0)

プロの不動産業者が顧客との商談中に「契約支援パッケージ」を即座に生成するためのガイドです。

## 利用可能なツール

| ツール名 | 説明 | 用途 |
|---|---|---|
| `generate_contract_support_package` | リスクマトリックス・価格交渉アンカー・推奨特約を生成 | 商談中の即時資料作成 |
| `assess_contract_risk` | 提案条項のリスクスコアとディールブレーカーを評価 | 契約内容の事前チェック |

## ダッシュボード

`?mode=contract` で契約専用画面が開きます。

- 区・町丁目・築年数・面積・価格・種別を入力
- 「契約支援パッケージ生成」ボタンで `generate_contract_support_package` を呼び出し
- 「リスク評価のみ」ボタンで `assess_contract_risk` を呼び出し

## 体験シナリオ

### シナリオ 1: 栄三丁目 4,200 万円マンション

**状況**: 中区栄三丁目の築28年マンション（72㎡）を 4,200 万円で売買契約予定。融資特約 14 日、検査なしのまま進めようとしている。

**手順**:
1. `?mode=contract` を開く
2. 区=`中区`、町丁目=`栄三丁目`、築年数=28、面積=72、価格=4200 を入力
3. 「契約支援パッケージ生成」をクリック
4. ツールが以下の内容を返す:
   - リスク: 洪水保険加入義務（高）、リノベ費用超過（高）
   - 推奨特約: 融資特約7日以内、建物状況調査必須、リノベ費用上限条項
   - 価格交渉アンカー: 成約相場 58 万円/㎡、リニア影響 +18〜25%
5. 顧客に「この特約を入れないとリスクが高い」と説明し、契約書に反映

### シナリオ 2: 中村区リニア駅周辺 5,800 万円区分所有

**状況**: 中村区名駅近くの築15年の区分所有オフィスを 5,800 万円で購入検討。将来のリニア開業を価格に反映したい。

**手順**:
1. 区=`中村区`、町丁目=`名駅一丁目`、築年数=15、面積=95、価格=5800
2. 「契約支援パッケージ生成」
3. ツールが以下を提示:
   - 将来価値: 2037 年リニア開業で +28〜35% 上昇見込み
   - 推奨: 「価格上昇条項（エスクロー）」を特約化し、開業後の価格上昇分を売主と共有
   - リスク: 低（全体スコア 28）
4. 顧客と「上昇分を 50% 買い手に還元する条項」を交渉し、契約に盛り込む

## 注意事項

- ツールの出力は既存公開データに基づく参考値です
- 最終的な契約判断は弁護士・宅建士にご相談ください
- Pro プラン以上でブランド入り PDF 出力が可能です（v6.9.1 以降で実装予定）
