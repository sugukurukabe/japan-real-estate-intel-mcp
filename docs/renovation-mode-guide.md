# リノベ利回り分析ガイド (v6.8.0)

名古屋市の不動産業者向けリノベーション利回り意思決定スイートの使い方ガイドです。

## 利用可能なツール

| ツール名 | 説明 | 用途 |
|---|---|---|
| `analyze_renovation_yield` | リノベ利回り分析 | 物件の取得〜リノベ〜賃貸/転売の利回りを算出 |
| `get_future_timeline` | 未来タイムライン | 2025-2050年の都市計画・インフラ・人口推移 |
| `get_chochou_profile` | 町丁目プロファイル | 地価・人口・進行中計画などの現状指標 |
| `recommend_renovation_targets` | 利回りランキング | 名古屋市全16区のリノベ利回りTop10 |

## ダッシュボード（MCP Apps UI）

`?mode=renovation` パラメータでリノベ専用画面に切り替わります。

4つのタブ:
- **未来**: 対象エリアの将来計画タイムライン
- **現状**: 地価・人口・世帯構成などの現状指標
- **リスク**: 浸水・地震・土砂災害リスク評価
- **収益性**: リノベ利回り計算結果

## 体験シナリオ

### シナリオ 1: 栄三丁目の築30年マンション

**状況**: 名古屋市中区栄三丁目で築30年のマンション（70㎡）が4,000万円で売りに出ている。リノベして賃貸に出すか検討中。

**手順**:
1. Claude Desktop で「名古屋市中区栄三丁目の築30年マンション70㎡を4000万円で買ってリノベしたら利回りはどうなる？」と質問
2. AI が `analyze_renovation_yield` を呼び出し、利回りを計算
3. 結果例:
   - 取得価格: 4,000万円
   - リノベ費用(中): 910万円 (130,000円/㎡ × 70㎡)
   - 月額賃料推定: 約18万円
   - 表面利回り: 約4.4%
   - 実質利回り: 約3.7%
   - 推奨戦略: 転売（利回りが6%未満のため）
4. 「この付近の未来計画は？」と追加質問 → `get_future_timeline` で栄地区再開発の影響を確認
5. 判断材料: 栄地区再開発により将来の地価上昇が見込まれるため、転売戦略で中長期保有も検討の余地あり

### シナリオ 2: リニア開業を見据えた中村区投資

**状況**: 2037年リニア中央新幹線開業に向けて、名古屋市中村区で仕入れる町丁目を3つ選びたい。

**手順**:
1. 「中村区でリニア開業を見据えて投資すべき町丁目を3つ提案して」と質問
2. AI が `get_future_timeline` で中村区の将来計画を確認（名古屋駅前再開発、笹島エリア等）
3. AI が `get_chochou_profile` で中村区の各町丁目の現状を調査
4. AI が `recommend_renovation_targets` でリノベ利回りを比較
5. 結果例:
   - **名駅一丁目〜三丁目**: リニア直結、再開発の中心。地価上昇+25%見込み
   - **太閤通**: 名駅至近で現在割安。リノベ後利回り7-8%が期待できる
   - **笹島(ささしま)**: ライブ24後続開発で注目。中長期の需要増+30%
6. 判断材料: 太閤通が利回りと将来性のバランスが最も良い

### シナリオ 3: 築古一戸建てリノベ向け利回り上位

**状況**: 築40年の一戸建てリノベに特化して、名古屋市全域で利回りの高い町丁目を探したい。

**手順**:
1. 「名古屋市で築40年一戸建て100㎡のリノベ利回りが高い町丁目Top10を見せて」と質問
2. AI が `recommend_renovation_targets` を呼び出し (buildingAge=40, floorArea=100, propertyType=house)
3. ダッシュボード（?mode=renovation）の「Top10ランキング」ボタンでも確認可能
4. 結果は表面利回り・実質利回り・総投資額・推奨戦略を含むランキング形式で表示
5. 上位の町丁目については、個別に `analyze_renovation_yield` で詳細を確認
6. 判断材料: 郊外区（守山区、緑区、港区）は地価が安く利回りが高いが、空室リスクも考慮が必要

## データソース

- **地価データ**: MLIT 不動産情報ライブラリ XIT001
- **人口・世帯**: e-Stat 令和2年国勢調査
- **都市計画**: 名古屋市オープンデータ + キュレーション
- **将来推計**: 社会保障・人口問題研究所 推計

## 注意事項

- 賃料推定は係数モデルによる推定値です（v6.9で実APIと接続予定）
- 利回り計算は簡易版であり、実際の投資判断には専門家の助言が必要です
- データは定期更新されますが、最新の市況を反映していない場合があります
