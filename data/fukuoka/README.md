## Sources

- Land Price: 国土交通省 地価公示 (福岡県抽出)
- Population: 国勢調査 令和2年・推計値 (e-Stat 国勢調査, 福岡市・北九州市統計)
- Flood: 福岡市ハザードマップ・国土交通省 ハザードマップポータル (那珂川/室見川/遠賀川/筑後川)
- Earthquake: 警固断層帯に関する地震調査研究 (地震調査研究推進本部), 福岡市地震ハザードマップ
- Municipalities: 国土数値情報 行政区域データ (福岡市7区+北九州市主要区+久留米市+春日市+大野城市)
- Neighborhoods: 国勢調査 小地域集計 (令和2年)

## Disclaimer

本データは代表的・概算値であり、実際の公示地価・国勢調査数値とは異なる場合があります。
