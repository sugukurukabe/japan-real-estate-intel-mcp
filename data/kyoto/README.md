## Sources

- Land Price: 国交省 地価公示 (京都府抽出)
- Population: 京都府統計書・各市区町村統計
- Flood: 国交省 ハザードマップポータル (鴨川/桂川/宇治川/木津川)
- Earthquake: 内閣府 花折断層帯地震被害想定・有馬高槻断層帯地震
- Municipalities: 国土数値情報 行政区域データ (京都市11区+宇治市+長岡京市+京田辺市)
- Neighborhoods: 国勢調査 小地域集計 (令和2年)

## Disclaimer

このデータは参考情報として提供するものです。実際の不動産取引・防災計画等への利用に際しては、最新の公式情報を参照し、専門家にご相談ください。地価・人口データは公示値・統計値に基づく推定値を含みます。浸水・地震リスクは最新のハザードマップをご確認ください。
