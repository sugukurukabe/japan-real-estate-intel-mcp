# Tokyo Data (東京都)

Bundled snapshot for the 23 special wards and major datasets used by `TokyoLoader`.

## Capabilities

| Capability | Available |
|---|---|
| Land price | YES |
| Transactions | YES |
| Population | YES |
| Flood / earthquake risk | YES |
| Municipalities | YES |
| Human flow | YES |
| Education (school districts) | YES |
| Corporate locations | YES |
| Crime stats | YES |
| PLATEAU 3D buildings | YES |
| Transport stations | YES |
| Commercial facilities | YES |
| Medical facilities | YES |
| Neighborhoods | YES |
| Zoning / vacancy / population projection / rosenka | YES |

詳細な列や出典は必要に応じて各 CSV / JSON の先頭コメントや `data/data-README.md` を参照してください。

## Sources (representative)

- Land price: 国交省 地価公示 (Tokyo extract)
- Population: 東京都統計年鑑
- Flood: 国交省 ハザードマップポータル
- Earthquake: 内閣府 首都直下地震被害想定
- Municipalities: 国土数値情報 行政区域データ (23区)
