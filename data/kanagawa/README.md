## Sources

- Land Price: 国交省 地価公示 (神奈川県抽出)
- Population: 神奈川県統計センター・横浜市・川崎市・相模原市統計
- Flood: 国交省 ハザードマップポータル (多摩川/鶴見川/相模川/境川/帷子川)
- Earthquake: 内閣府 相模トラフ巨大地震被害想定・首都直下地震
- Municipalities: 国土数値情報 行政区域データ (横浜市・川崎市各区+主要市)
- Neighborhoods: 国勢調査 小地域集計 (令和2年)

## Disclaimer

本データは教育・研究目的のサンプルデータです。実際の不動産取引・防災計画等の判断には使用しないでください。地価・人口・ハザード情報は各公的機関の最新情報をご確認ください。
