## Sources

- Land Price: 国交省 地価公示 (大阪府抽出)
- Population: 大阪府統計年鑑・各市区町村統計
- Flood: 国交省 ハザードマップポータル (淀川/大和川/道頓堀川/寝屋川/神崎川)
- Earthquake: 内閣府 南海トラフ巨大地震被害想定・上町断層帯地震
- Municipalities: 国土数値情報 行政区域データ (大阪市24区+堺市+主要市)
- Neighborhoods: 国勢調査 小地域集計 (令和2年)
