## Sources

- Land Price: 国交省 地価公示 (兵庫県抽出)
- Population: 兵庫県統計年鑑・各市区町村統計
- Flood: 国交省 ハザードマップポータル (武庫川/猪名川/明石川/加古川)
- Earthquake: 内閣府 南海トラフ巨大地震被害想定・六甲・淡路断層帯地震（1995年阪神淡路大震災震源断層帯）
- Municipalities: 国土数値情報 行政区域データ (神戸市8区+姫路市+西宮市+尼崎市+明石市+宝塚市)
- Neighborhoods: 国勢調査 小地域集計 (令和2年)

## Notes

兵庫県は1995年1月17日の阪神・淡路大震災（M7.3）の被災地であり、六甲・淡路断層帯が走る地震リスクの高い地域です。
地震リスクデータは内閣府・兵庫県の公表資料に基づく推計値であり、実際の被害を保証するものではありません。
不動産投資判断に際しては必ず最新の公的ハザードマップ及び専門家の意見を参照してください。
