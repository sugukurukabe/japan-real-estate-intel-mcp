## Sources

- Land Price: 国交省 地価公示 (北海道抽出)
- Population: 北海道統計年鑑・各市区町村統計
- Flood: 国交省 ハザードマップポータル (石狩川/豊平川/十勝川/空知川)
- Earthquake: 内閣府 千島海溝沿い巨大地震被害想定・北海道胆振東部地震教訓
- Municipalities: 国土数値情報 行政区域データ (札幌市10区+函館・旭川・小樽・釧路・帯広)
- Neighborhoods: 国勢調査 小地域集計 (令和2年)
