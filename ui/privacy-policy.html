Privacy Policy / プライバシーポリシー

Effective: 2026-07-22 — Last updated: 2026-07-22

1. Operator

Japan Real Estate Intel MCP ("Service") is operated by 株式会社スグクル (Sugukuru Inc.) ("we").
Contact: info@sugu-kuru.co.jp or via GitHub Issues at github.com/sugukurukabe/japan-real-estate-intel-mcp

2. Data We Collect

  • No authentication required: This is a public, authless connector. We do not use OAuth and do not require an account.
  • MCP Session Data: A random UUID per session. No persistent user accounts.
  • License Keys (Pro/Enterprise, optional): If you supply a license key for paid tiers, it is verified by cryptographic signature and not stored server-side beyond the record created at purchase time (linked to your Stripe checkout session, not your MCP usage).
  • Server Logs: IP address (via reverse proxy), request path, user-agent, timestamps. Retained for 30 days for operational purposes.
  • API Keys: If you provide an API_KEY for authentication, it is compared in-memory. We do not log API key values.

3. Data We Do NOT Collect

  • Names, email addresses, or phone numbers
  • Conversation content between you and the AI model
  • Financial or payment information
  • Location data beyond what you explicitly provide as tool inputs

4. How We Use Data

  • Session management and rate limiting
  • Error monitoring and service reliability
  • Abuse prevention

5. Third-Party Services

  • Government APIs (MLIT, e-Stat): We relay queries to Japanese government open-data APIs. No personal data is sent.
  • Google Maps Platform (optional): If GOOGLE_MAPS_API_KEY is configured, the assess_exterior_visuals and analyze_commute_accessibility tools send the property's approximate coordinates to the Street View Static API and Distance Matrix API to fetch a street-level image and compute commute times. No account identifiers accompany these requests. See Google Privacy Policy.
  • Google Gemini (optional): If GOOGLE_GENAI_API_KEY is configured, area analysis prompts (and, for assess_exterior_visuals, the Street View image above) are sent to Google's Gemini API for narrative generation / vision analysis. See Google Privacy Policy.
  • Sentry (optional): If SENTRY_DSN is configured, error reports may include stack traces and request metadata.
  • Tile servers: Map tiles are loaded from OpenStreetMap and CARTO CDNs. See their respective privacy policies.

6. Data Retention

MCP sessions expire after 30 minutes of inactivity. Server logs are retained for 30 days. Downloadable artifacts (PDF/Excel/CSV/Markdown reports generated by tools like generate_area_report) are retained for 24 hours by default (ARTIFACT_TTL_HOURS) and then permanently deleted from the server.

7. Your Rights (GDPR / APPI)

Since we collect minimal personal data, most GDPR/APPI rights are satisfied by default. You may request deletion of server logs referencing your IP address by contacting us via GitHub Issues. We will respond within 30 days.

8. Security

We use HTTPS (TLS 1.3 via Caddy), helmet security headers, rate limiting, and optional API key authentication. See our Security Policy.

9. Children

This Service is not directed to children under 16. We do not knowingly collect data from minors.

10. Changes

We may update this policy. Changes will be posted here with an updated effective date. Continued use constitutes acceptance.
